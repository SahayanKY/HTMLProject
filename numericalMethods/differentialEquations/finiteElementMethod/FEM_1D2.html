さて、前回の最後に言及したが、関数はあまりに自由度が高いので、既知の関数、基底関数 \(\phi_i ,\psi_i\)の組み合わせで近似することにする。 \begin{align} f(x) &= f_0 (x) +\sum_i f_i \phi_i (x) \\ % % w(x) &= \sum_i w_i \psi_i (x) \end{align} \end{align} 一方で\(f_0\)はディリクレ境界条件を満たす関数とする。従って今回は \begin{align} f_0 (x) = 0 \quad (0\leq x \leq 1) \end{align} とする。この近似式を弱形式に直接代入する。 \begin{align} \int_0^1 \frac{\d w}{\d x} \frac{\d f}{\d x} \d x + \\ % % \int_0^1 \left( \sum_i w_i \frac{\d \psi_i}{\d x} \right) \cdot \left( \sum_j f_j \frac{\d \phi_j}{\d x} \right) \d x + \int_0^1 \sum_i w_i \psi_i (x) \d x &= 0 \\ % % \sum_i \sum_j w_i f_j \int_0^1 \frac{\d \psi_i}{\d x} \frac{\d \phi_j}{\d x} \d x + \sum_i w_i \int_0^1 \psi_i (x) \d x &= 0 \\ % % \sum_i w_i \left( \sum_j f_j \int_0^1 \frac{\d \psi_i}{\d x} \frac{\d \phi_j}{\d x} \d x + 全て0である。即ち、 \begin{align} \sum_j f_j \int_0^1 \frac{\d \psi_i}{\d x} \frac{\d \phi_j}{\d x} \d x + \int_0^1 \psi_i (x) \d x = 0 \end{align} となる。 \begin{align} a_{ij} &= \int_0^1 \frac{\d \psi_i}{\d x}
